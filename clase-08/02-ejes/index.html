Ejes -- FLEX CONTAINER

Cada vez que se define un flex-container este adquiere automaticamente dos ejes :

  • Main axis (principal,horizontal por default)
  • Cross axis (secundario,vertical por default)

De forma predeterminada los elementos (Flex-items) se alinean respecto al main axis.

La direccion de los ejes puede cambiarse con la propiedad flex-direction(aplicada al container)

		.container-demo{
			display:flex;
			flex-direction:row-reverse;
		}

Los valores para direction son :

  • row(por default left to rigth)
  • row-reverse(rigth to left)
  • column (top to bottom)
  • column-reverse (bottom to top)

Alinear elementos respecto al main-axis

Se usa la propiedad justify-content y tienen los siguientes valores :

  • flex-start (al inicio)
  • flex-end (al final )
  • center (al medio)
  • space-between (divide el espacio sobrante y lo distribuye entre los items
  • space-around (divide el epsacio sobrante y lo distribuye alrededor de los items

  • flex-start
